### GLB and LUB:
For an ordered set $\mathbb{X}$, if we take any bounded above subset $\mathbb{Y} \subset \mathbb{X}$, then if LUB ($\mathbb{Y}) \in \mathbb{Y}$, then we say $\mathbb{X}$ has LUB property.
